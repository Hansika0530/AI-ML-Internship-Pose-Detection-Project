# AI/ML Internship - Pose Detection Project

# Output Preview
The project successfully detects human body landmarks in real time using the webcam and displays a pose skeleton on the screen. The system tracks facial points, shoulders, arms, and upper body posture using AI-powered pose estimation.Shows Pose detected or not on the screen.

# Features Demonstrated

- Real-time webcam pose detection
- AI landmark tracking using MediaPipe
- Skeleton structure visualization
- Live FPS display for performance monitoring
- Pose Detected status indicator
- Human body keypoint recognition

## Sample Output

The application identifies body landmarks such as:
- Nose
- Eyes
- Shoulders
- Arms
- Upper body joints
